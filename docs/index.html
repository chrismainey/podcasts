Overview

This site attempts to list Data Science and general Science (with a Stats focus) podcasts. Please feel free to add any missing group or conference. To propose a change, just click the pencil icon in the top left hand corner.

This list is maintained by Jumping Rivers. See

  • meetingsR for a list of R events and user groups.
  • meetingsDS for a list of Data science events and user groups.
